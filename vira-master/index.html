Путешествие по Лене

Тур по одной из самых длинных рек России — Лене, где вы увидите уникальные природные образования и насладитесь спокойствием величественных пейзажей. Экспедиция по Лене — это возможность увидеть нетронутую природу Сибири.

Загадки реки Амур

Путешествие по реке Амур предлагает уникальную возможность исследовать одну из самых загадочных рек России. Вы увидите удивительную природу и узнаете множество легенд, связанных с этой великой рекой.

По рекам Кавказа

Маршрут по горным рекам Кавказа предлагает насладиться бурлящими водами и живописными ущельями. Этот тур для тех, кто ищет активные приключения и незабываемые впечатления среди горных вершин.

Озеро Байкал

Путешествие по рекам, впадающим в озеро Байкал, подарит вам возможность исследовать одно из самых чистых озёр мира. Вы увидите кристально чистую воду, потрясающие пейзажи и богатую природу Сибири.

Экспедиция по Оби
